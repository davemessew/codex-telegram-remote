# Publishing

The public repository slug is:

```text
davemessew/codex-telegram-remote
```

The plugin card points at:

```text
https://github.com/davemessew/codex-telegram-remote
```

For a public GitHub install, users run:

```powershell
codex plugin marketplace add https://github.com/davemessew/codex-telegram-remote
codex plugin add codex-telegram-remote@codex-telegram-remote
```

Keep `marketplace.json` at `.agents/plugins/marketplace.json` and the plugin package at `plugins/codex-telegram-remote/`.
